1. open terminal
  $ jekyll new blogName.github.io
  $ cd blogName.github.io
  $ bundle exec jekyll serve

2. browse to http://localhost:4000

3. open terminal
  $ cd ..
  $ code blogName.github.io

4. create README.md

5. open git hub

6. create repo blogName.github.io
